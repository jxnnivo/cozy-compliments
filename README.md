# Cozy Compliment Generator

A lightweight interactive web application that displays randomized cozy affirmations with a smooth fade animation.

Designed as a small frontend project to practice DOM manipulation, event-driven programming, and UI transitions using vanilla JavaScript.

## Features

- Random compliment generation
- Smooth fade-out / fade-in animation
- Event-driven button interaction
- Responsive centered layout
- Clean, aesthetic UI design

## Technologies Used

- HTML
- CSS
- JavaScript

## What I Practiced

- DOM element selection
- Event listeners
- Array-based random selection
- UI state management with CSS transitions
- setTimeout() for controlled animation timing
- Clean project structure and code documentation
